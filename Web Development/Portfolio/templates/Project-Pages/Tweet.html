004 : Offensive Tweet Detection

This project involved building a model that can accurately classify and identify a wide spectrum of tweets as either being hate speech or not. Encodings and attention masks were generated for each tweet before padding the input sequences so that the BERT backbone classification model only pays attention to relevant tokens in the tweet. This resulted in the model achieving a validation accuracy of almost 96%. To view the project, please visit this link.
